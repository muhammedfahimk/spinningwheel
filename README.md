# Lucky Draw Wheel

A simple real-time Lucky Draw Wheel application built using **Flask**, **Flask-SocketIO**, **SQLite**, HTML, CSS, and JavaScript.

## Features

- Real-time spinning wheel using SocketIO
- Random winner selection
- Selected members are saved in SQLite database
- Already selected members are removed from the next draw
- Reset option to make all members available again
- Admin page to view member selection status

## Tech Stack

- Python
- Flask
- Flask-SocketIO
- SQLite
- HTML
- CSS
- JavaScript
